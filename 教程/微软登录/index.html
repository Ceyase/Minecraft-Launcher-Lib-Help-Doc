微软登录

使用微软帐户登录需要Azure应用程序。

创建Azure应用程序

要登录Microsoft，您需要先创建Azure应用程序。查看 这个教程 去创建一个. 您需要新应用程序的客户端ID、客户端密码和重定向URL。

让用户登录

用户需要在浏览器中登录。这可以是IE浏览器，也可以是嵌入在程序中的浏览器。要获取用于登录的网址，请使用 minecraft_launcher_lib.microsoft_account.get_login_url(client_id, redirect_uri)打开URL并测试是否可以登录。登录后，您将被重定向到 https://?code=代码&state=<optional. 代码就是你需要的密钥，你能使用 minecraft_launcher_lib.microsoft_account.get_auth_code_from_url(url) 从网址获取代码. 你也可以使用minecraft_launcher_lib.microsoft_account.url_contains_auth_code(url: str) 检查给定URL是否包含代码。

进行登录

使用 minecraft_launcher_lib.microsoft_account.complete_login(client_id: str, client_secret: str, redirect_uri: str, auth_code: str) 登录到Minecraft。auth_code是您在上一步中从网址里获得的代码。您将得到以下结果：

{
    "id" : "UUID",
    "name" : "用户名",
    "access_token": "access令牌",
    "refresh_token": "refresh令牌",
    "skins" : [{
        "id" : "6a6e65e5-76dd-4c3c-a625-162924514568",
        "state" : "ACTIVE",
        "url" : "http://textures.minecraft.net/texture/1a4af718455d4aab528e7a61f86fa25e6a369d1768dcb13f7df319a713eb810b",
        "variant" : "CLASSIC",
        "alias" : "STEVE"
    } ],
    "capes" : []
}

正如您所看到的，它包含了您使用get_minecraft_command()需要的所有选项。

刷新

如果你想刷新玩家数据，你只需要使用 minecraft_launcher_lib.microsoft_account.complete_refresh(client_id: str, client_secret: str, redirect_uri: str, refresh_token: str) 刷新令牌。如果刷新失败，它将抛出InvalidRefreshToken异常。在这种情况下，您可能需要用户再次登录。
