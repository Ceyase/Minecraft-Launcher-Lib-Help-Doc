自动安装Forge

本教程演示如何使用MLL安装forge。请注意，Forge开发人员不希望启动器自动安装Forge，除非您捐赠他们钱。

获取Forge版本

在安装Forge之前，您需要知道都有哪些Forge版本。请使用 find_forge_version()

forge_version = minecraft_launcher_lib.forge.find_forge_version("1.17.1")
if forge_version is None:
    print("这个Minecraft版本没有所对应的Forge")
    return

在本案例中，我们得到了1.17.1的最新Forge版本。

安装Forge版本

现在我们有了Forge版本，可以开始安装了。请使用 install_forge_version()

minecraft_launcher_lib.forge.install_forge_version(forge_version, minecraft_directory)

install_forge_version() 支持和 install_minecraft_version()一样的回调. install_forge_version() 不支持1.12之前的版本.

使用 supports_automatic_install 检查此功能是否支持您的Forge版本。

启动Forge版本

很不幸, find_forge_version() 获取的版本无法用于 get_minecraft_command(), 因为forge安装后的游戏名称稍有不同. 请使用 get_installed_versions() 来获取启动的正确版本id。
