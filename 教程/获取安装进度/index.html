获取安装进度

安装新的Minecraft版本可能需要一些时间，这取决于我们的网速。最好是向用户显示进度，例如在进度栏中。

为了告诉您的程序当前的进度，MLL可以使用回调。回调只是您编写的普通函数，它可以让MLL调用。以下是一个例子：

import minecraft_launcher_lib

# 这里是获取进度条的最大值（长度）
current_max = 0

# 这里定义了一个函数，获取当前的状态
def set_status(status: str):
    print(status)

# 这个函数获取了当前的进度
def set_progress(progress: int):
    if current_max != 0:
        # 这条函数打印出了当前进度/最大值
        print(f"{progress}/{current_max}")

# 这个函数获取了最大值
def set_max(new_max: int):
    # 定义一个全局变量
    global current_max
    # 赋值
    current_max = new_max

# 这个函数获取了默认的.minecraft文件夹路径
minecraft_directory = minecraft_launcher_lib.utils.get_minecraft_directory()

# 这个dict设定了回调的内容，全部是刚才设定的函数
callback = {
    "setStatus": set_status,
    "setProgress": set_progress,
    "setMax": set_max
}

# 下载1.17并启用回调
minecraft_launcher_lib.install.install_minecraft_version("1.17", minecraft_directory, callback=callback)

正如您所看到的，回调是一个带有函数的dict。这些函数由您定义。您可以在这些函数中编写任何您想要的内容。在上面的示例中，它将当前状态打印到命令行。
