安装

这有两种途径去安装Minecraft Launcher Lib。

  • pip

与其他的Python库一样，MLL库可以直接从 PyPI 下载. 我推荐你使用这种方式去安装。pip install -U minecraft-launcher-lib

  • 从源代码安装

您还可以直接从源代码安装MLL。这可以让你最快的体验到新的模块。但如果你不需要这些模块，不推荐这样做。pip install -U git+https://gitlab.com/JakobDev/minecraft-launcher-lib.git
